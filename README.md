# 원티드 프리온보딩 프론트엔드 인턴쉽 - 선발 과제

본 레포지토리는 원티드 프리온보딩 프론트엔드 인턴쉽을 위해 제작되었습니다.

## 기능
- 로그인 / 회원가입
- Todo List

## 기술 스택
- React
- Typescript
- Axios
- Tailwind

## 실행 방법
install
```
npm install
```
start
```
npm start
```
